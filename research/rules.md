# Research and Analysis Rules

Rules for AI agents when conducting research, analyzing information, and synthesizing findings.

## Research Principles

### Objectivity
- Approach topics without bias
- Consider multiple perspectives
- Separate facts from opinions
- Acknowledge your own assumptions
- Report findings fairly

### Thoroughness
- Search comprehensively
- Use multiple sources
- Verify key information
- Look for contradictory evidence
- Document your search process

### Quality
- Evaluate source credibility
- Prioritize primary sources
- Use recent, relevant information
- Check for updates and corrections
- Cite all sources properly

## Research Process

### 1. Define the Question
- Clarify what you're researching
- Identify specific sub-questions
- Establish scope and boundaries
- Define success criteria

### 2. Gather Information
- Search multiple sources
- Use appropriate databases
- Take organized notes
- Track source information
- Save relevant materials

### 3. Evaluate Sources
- Check author credentials
- Verify publication reputation
- Assess methodology
- Look for peer review
- Consider potential biases

### 4. Analyze Findings
- Identify patterns and themes
- Note contradictions
- Assess evidence strength
- Draw logical conclusions
- Document reasoning

### 5. Synthesize and Report
- Organize findings clearly
- Support claims with evidence
- Acknowledge limitations
- Provide actionable insights
- Include source citations

## Source Evaluation

### Credibility Criteria
| Factor | Questions to Ask |
|--------|-----------------|
| Authority | Who created this? What are their credentials? |
| Accuracy | Is the information verifiable? Are sources cited? |
| Objectivity | Is there bias? Is it balanced? |
| Currency | When was it published? Is it still relevant? |
| Coverage | How comprehensive is it? |

### Source Hierarchy
1. **Primary sources**: Original research, official documents
2. **Secondary sources**: Analysis of primary sources
3. **Tertiary sources**: Summaries, encyclopedias
4. **Expert opinions**: Subject matter expert statements
5. **General sources**: News, general publications

## Analysis Methods

### Comparative Analysis
- Identify comparison criteria
- Gather data for each subject
- Create comparison matrix
- Highlight differences and similarities
- Draw conclusions

### Trend Analysis
- Identify time periods
- Gather historical data
- Plot changes over time
- Identify patterns
- Project future trends

### SWOT Analysis
- **S**trengths: Internal advantages
- **W**eaknesses: Internal disadvantages
- **O**pportunities: External favorable factors
- **T**hreats: External unfavorable factors

### Root Cause Analysis
1. Define the problem
2. Collect data
3. Identify possible causes
4. Test causes
5. Identify root cause
6. Recommend solutions

## Research Output Format

### Research Summary
```markdown
# Research: [Topic]

## Research Question
Clear statement of what was investigated.

## Methodology
How the research was conducted.

## Key Findings
1. Finding 1 with supporting evidence
2. Finding 2 with supporting evidence
3. Finding 3 with supporting evidence

## Analysis
Interpretation of findings.

## Conclusions
What the findings mean.

## Recommendations
Suggested actions based on research.

## Sources
1. Source 1
2. Source 2

## Appendix
Supporting materials.
```

## Data Analysis Rules

### Data Quality
- Verify data accuracy
- Check for completeness
- Identify outliers
- Understand data sources
- Document data issues

### Statistical Rigor
- Use appropriate methods
- Report uncertainty
- Avoid cherry-picking
- Consider sample size
- Be aware of limitations

### Visualization
- Choose appropriate chart types
- Label clearly
- Use consistent scales
- Avoid distortion
- Include context

## Reporting Findings

### Be Transparent
- Show your work
- Acknowledge limitations
- Discuss uncertainties
- Note what you couldn't find

### Be Actionable
- Provide clear conclusions
- Make specific recommendations
- Prioritize insights
- Suggest next steps

### Be Honest
- Report negative findings
- Don't overstate certainty
- Distinguish correlation from causation
- Acknowledge alternative interpretations
